
# MARUTI Primary Care Website Version 2.0

Includes:
- Public pages: Home, About, Services, Membership, Weight Loss, Medicare, Contact
- Three primary CTAs: Become a Member, Patient Login, Book Introductory Consultation
- Signup page
- Login page
- Protected-forms concept: Patient Forms requires signup/login
- Enrollment workflow: membership selection, registration, medical history, consent, e-signature, Stripe placeholder
- Future Patient Portal dashboard mockup
- $250 one-time registration fee
- Media placeholders and video section

Important production notes:
- Replace demo signup/login with Clerk, Auth0, or another secure authentication provider.
- Connect enrollment forms to a HIPAA-conscious backend/workflow.
- Use Stripe Checkout/Billing for payments; do not store credit-card numbers directly.
- Have legal agreements reviewed by a Massachusetts healthcare attorney.
